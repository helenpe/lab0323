3. 상세 제품 정보: CODEBOX (폐쇄망 설치형 AI 코드 개발 어플라이언스)
- 개요: 내부망에서 작동하는 코드 생성형 AI봇, LLM 기반 멀티 에이전트 인터페이스 솔루션입니다.
- 특징: 
  * 폐쇄망 환경에서도 실시간 코드 개발 지원
  * 믿을 수 있는 설치형 보안 솔루션
  * 저렴한 GPU 기반 서버 + 저비용 어플라이언스 + 전용 LLM 탑재

- 주요 고객군: 
  * 보안 규제가 강하거나 내부 AI 개발 환경 구축이 필요한 금융·공공·의료·방위 기관

- 핵심가치:
  * 보안 준수: 폐쇄망 내 설치로 코드 유출 걱정 무
  * 개발 생산성 향상: 코드 작성, 분석, 테스트 자동화로 프로젝트 가속화
  * 경제적 비용 확보: 낮은 비용의 운영 인프라 구축 지원

- 주요기능:
  * 코드 자동 생성: 고품질 코드 생성 및 자동 완성 지원
  * 코드 분석/최적화: 코드 리펙토링 제안 및 분석 보고서 자동 생성
  * 코드 문서화 생성: 스스로 라이브러리/코멘트 추가 기능
  * 전용 LLM 지원: 보안 특화 Fine-Tuned 전용 모델 사용
  * IDE 연동: 사용중인 개발 플러그인 연동 실시간 개발 보조
  * 자산 보안 보존: 모든 코드·지적재산은 조직 내 설치된 장치 내에만 저장

- 특장점:
  1) 폐쇄망 특화 설치형 패키지: GPU 기반 Workstation/Server 형태로 기업 내부 직접 설치.
  2) AI 개발 에이전트 일체 내장: Code Assistant, Data Analyst 등 특화 Agent 기본 탑재.
  3) 코드 전용 LLM 내장: kt ds Fine-Tuned 전문 모델로 오픈소스 대비 약 5배 성능 지원.
  4) AI 가공 품질·거버넌스 형성: 코드 작성 표준 준수 자동 분석 및 품질 관리.
  5) 환경 제약 최소화: 자체 서버만으로 AI 개발 생산성 확보 가능.

- 활용 시나리오:
  * 보안이 중요한 프로젝트에서 내부망 AI 코딩 도구 사용 희망 시
  * 공공/금융 프로젝트에서 코드 품질 및 규제 관리가 필요한 경우
  * IT 자산 유출을 최소화하며 개발 생산성을 높여야 하는 경우

4. 고객 사례
- 사례 1: S사 금융지능팀 (공공·금융기관)
  * Challenge: 데이터 외부 유출 보전 이슈
  * Solution: 폐쇄형 서버에 AI봇 구축 + 코드 자동 생성
  * Result: 코드 생성 시간 기존 대비 30% 감소
- 사례 2: 중앙부처 B (공공)
  * Challenge: 코드 품질 표준화 및 보안 감사 대비 부족
  * Solution: CodeBox의 내부 자산 전담 코드 관리 지원
  * Result: 코드 리뷰 시간 60% 단축, 품질 표준성 확보

5. 요금제
- CodeBox Standard: Workstation GPU 탑재 + 전용 LLM
- CodeBox Enterprise: Server급 GPU 탑재 + 커스텀 LLM + 전문가 지원
